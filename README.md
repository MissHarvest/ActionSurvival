### 🎮 플레이 방법

이동: WASD

달리기: Left Shift
  
인벤토리: I

아이템 사용 : 아이템 슬롯 클릭 후 나타나는 버튼에서 선택

상호작용: E
- 자원 채집(채광, 벌목)

***
### 🛠 구현 기능

##### 1. 자원 수집
- 벌목 및 채광을 통해 자원(나무, 돌) 획득

##### 2. 플레이어 생존 관리 시스템
- 체력
  
- 허기
  시간이 지남에 따라서 점차 감소
  일정 수치 이상일 경우 체력이 점차 회복
  0 이 될 경우 체력이 점차 감소

##### 3. 인벤토리
- 아이템 장착 / 해제
- 아이템 버리기

##### 4. UI
- 플레이어 컨디션 (체력, 허기)
- 인벤토리
